## Best practises

- **Mimic the content:**  Placeholders should be used to represent the structure of the content while it is loading.
They should match the layout and proportions of the final content, maintaining consistency in spacing and alignment.
- **Keep it lightweight:** Avoid creating overly detailed placeholder designs. Use placeholders to cover essential content only,
nothing too complex. This ensures a smooth transition once the actual content loads.
