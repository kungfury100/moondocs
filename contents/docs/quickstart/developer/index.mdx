    ## Installation

    - To get variables from Figma we use Variables REST API endpoints
    - Create `.env` file and add the necessary information:

    ```bash
    # Personal access token
    FIGMA_TOKEN=figd_personal-access-token

    # File ID (figma.com/design/file-id/file-name)
    FIGMA_FILE=your-brand:BZiHkvF7pXFHrFH8P0cG2T
    FIGMA_COMPONENTS_FILE=your-brand:S3q1SkVngbwHuwpxHKCsgtJj

    # Installation directory (optional)
    MOON_OUTPUT_DIR=assets
    ```

    ## Usage

    - Installation, this generates `css/your-brand-base.css` with all CSS variables.

    ```
    npx @heathmont/moon-css
    ```

    - Generate Tailwind preset, this adds `js/your-brand-base-preset.js` for Tailwind utility classes (optional).

    ```
    npx @heathmont/moon-css --with-preset
    ```

    - Include pre-built components, this generates `js/moon-components.js` with ready-to-use component classes (optional).

    ```
    npx @heathmont/moon-css --with-preset --with-components
    ```

    ## Install components

    ```bash
    # Using npm
    npm install @heathmont/moon-core-tw

    # Using pnpm
    pnpm install @heathmont/moon-core-tw

    # Using yarn
    yarn add @heathmont/moon-core-tw
    ```

    ### Usage

    ```tsx
    import { Button, Input, Modal } from "@heathmont/moon-core-tw/lib";

    const App = () => {
      const [open, setOpen] = useState(false);

      return (
        <div>
          <Input placeholder="Enter text..." />
          <Button onClick={() => setOpen(true)}>Open Modal</Button>
          {open && (
            <Modal onClose={() => setOpen(false)}>
              <p>This is a modal!</p>
            </Modal>
          )}
        </div>
      );
    };

   export default App;
    ```

    ## Install themes

    ```bash
    # Using npm
    npm install @heathmont/moon-themes-tw

    # Using pnpm
    pnpm install @heathmont/moon-themes-tw

    # Using yarn
    ```
